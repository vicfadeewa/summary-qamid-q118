## Перечень автоматизируемых сценариев ##
**Позитивные сценарии**
1. Открытие формы оплаты по карте после клика на кнопку "Купить"
Ожидаемый результат:
Форма оплаты по карте отображается на экране
2. Успешная оплата тура при валидном заполнение полей формы "Купить" с APPROVED картой
Ожидаемый результат:
Вывод сообщения с текстом об успешной оплате
3. Отказ в оплате тура при валидном заполнение полей формы "Купить" с DECLINED картой
Ожидаемый результат: Вывод сообщения с текстом об ошибке

**Негативные сценарии для формы "Купить"**
1. Ввести меньше 16 цифр в поле "Номер карты". Поля месяц, год, владелец, CVC/CVV заполнить валидными данными

2. Заполнить поле "Номер карты" спецсимволами и буквами. Поля месяц, год, владелец, CVC/CVV заполнить валидными данными

3. Оставить поле "Номер карты" пустым. Поля месяц, год, владелец, CVC/CVV заполнить валидными данными

4. Ввести в поле "Месяц" число больше 12. Поля номер карты, год, владелец, CVC/CVV заполнить валидными данными

5. Ввести в поле "Месяц" буквы. Поля номер карты, год, владелец, CVC/CVV заполнить валидными данными

6. Оставить поле "Месяц" пустым. Поля номер карты, год, владелец, CVC/CVV заполнить валидными данными

7. Ввести имя в поле "Владелец" на кириллице. Поля номер карты, год, месяц, CVC/CVV заполнить валидными данными

8. Ввести цифры в поле "Владелец". Поля номер карты, год, месяц, CVC/CVV заполнить валидными данными

9. Оставить поле "Владелец" пустым. Поля номер карты, год, месяц, CVC/CVV заполнить валидными данными
